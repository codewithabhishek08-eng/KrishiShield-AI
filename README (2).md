## 🌍 The Problem

India has **146 million smallholder farmers** — they feed a nation yet remain the most economically vulnerable segment of its population.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    THE FARMER'S CRISIS TRIANGLE                     │
│                                                                     │
│         PRICE VOLATILITY              DISEASE OUTBREAK              │
│         ───────────────               ───────────────               │
│         Tomato: ₹2/kg → ₹80/kg       Early Blight destroys         │
│         in 6 weeks. No warning.       30-40% yield overnight.       │
│         No protection.                No early detection.           │
│                    ╲                 ╱                              │
│                     ╲               ╱                               │
│                      ▼             ▼                                │
│                   CREDIT TRAP                                       │
│                   ───────────                                       │
│                   Local lender: 15-36% interest                     │
│                   Bank loan: 6-month process                        │
│                   Result: Generational debt                         │
└─────────────────────────────────────────────────────────────────────┘
```

| Pain Point | Current Reality | Annual Impact |
|-----------|----------------|---------------|
| 💸 Price Crashes | Farmers sell at harvest when prices are lowest | ₹75,000 Cr annual loss |
| 🦠 Disease Spread | Detected visually when 30-40% crop is already lost | 18% yield loss nationwide |
| 💳 Credit Gap | 45% of farmers rely on informal lenders at 24%+ interest | ₹8.3 Lakh Cr trapped in debt |
| 📡 Data Blindness | No real-time field intelligence available affordably | 0 decisions are data-driven |

## ✨ Core Features

### 🧠 Farmer Dashboard — Mission Control
- **Crop Health Score** — Real-time FICO-style field health index (0–100) from satellite + weather + soil telemetry
- **Kisan AI Assistant** — Groq-powered conversational AI in Hindi/English, answers every farming question
- **Crop Diagnostics** — Image upload + symptom description → AI disease/pest identification + treatment protocol
- **Live Weather Overlay** — Hyperlocal forecast integrated with crop-stage advisory
- **Quick Actions** — One-tap access to price lock, loan application, satellite scan, expert connect

### 📈 Market Intelligence
- **200-Crop Database** — 50 fruits, 50 vegetables, 50 field crops, 50 seeds with live mandi prices
- **6-Month Price Forecast** — ML-powered prediction with confidence bands
- **Smart Contract Price Lock** — Blockchain-backed guaranteed price regardless of market crash
- **Supply Chain Alerts** — Regional surplus/deficit signals affecting price
- **Mandi Price Tracker** — City-specific real-time prices matching farmer's saved location

### 🛰️ Satellite Field Intelligence
- **NDVI Heatmap** — Live vegetation index canvas painted row-by-row from satellite data
- **Disease Probability Map** — AI-predicted outbreak zones with community alert overlay
- **Yield Forecast vs Regional Average** — Visual comparison with treatment/no-treatment projection
- **Rainfall Correlation** — Water deficit/surplus mapped against crop water need
- **Global Field Selector** — Leaflet.js world map with country → state → village drill-down

### 💳 Fintech & Loans
- **Harvest-Backed Micro Loans** — Instant credit against verified upcoming yield
- **1% vs 15% Comparison** — Live calculator showing savings over traditional lending
- **Government Scheme Navigator** — PM-FASAL, PM-Kisan, eNAM, KCC — eligibility check + application guide
- **Loan EMI Calculator** — Real-time slider-driven calculation with animated reveal

### 👤 Profile & Personalisation
- **Location-Aware Intelligence** — All data, prices, alerts, and AI advice scoped to saved city/state
- **Multi-Crop Management** — Track multiple crops simultaneously
- **Diagnosis History** — Persistent record of all past AI diagnoses
- **Expert Connect** — Direct message to agricultural specialists

## 🛰️ Satellite Intelligence

### NDVI Processing Pipeline

```
Sentinel-2 Imagery
        │
        ▼
┌───────────────────┐
│  Band Extraction  │──── NIR (Band 8) + Red (Band 4)
└────────┬──────────┘
         │
         ▼
┌───────────────────┐
│  NDVI Calculation │──── NDVI = (NIR - Red) / (NIR + Red)
└────────┬──────────┘
         │
         ▼
┌───────────────────┐     Green  (#4CAF50) → Healthy  0.6–1.0
│  Colour Mapping   │──── Amber  (#F57F17) → Moderate 0.3–0.6
└────────┬──────────┘     Red    (#B71C1C) → Stressed 0.0–0.3
         │
         ▼
┌───────────────────┐
│  Canvas Render    │──── Row-by-row paint, 2ms stagger
│  + Groq Analysis  │──── AI interprets score in context
└───────────────────┘
```

## 📊 Market Intelligence

### Smart Contract Price Lock Flow

```
Farmer selects crop + quantity + target date
            │
            ▼
    AI validates market conditions
            │
            ▼
    Smart contract parameters set
    (price, quantity, date, farmer ID)
            │
            ▼
    Contract deployed to blockchain
            │
            ▼
    Hash generated + stored locally
            │
            ▼
    At harvest: automatic settlement
    regardless of market price
```
